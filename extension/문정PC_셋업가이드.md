# 채널에이전트 설치 가이드 (문정 PC)

## 사전 조건
- Chrome 브라우저가 설치되어 있어야 합니다
- GitHub에서 최신 코드를 받을 수 있어야 합니다

---

## 1단계: 확장 파일 로컬 복사

Google Drive 경로에서 직접 로드하면 크롬 재시작 시 삭제되므로 **로컬에 복사**합니다.

1. `C:\extension\` 폴더 생성
2. GitHub에서 받은 `extension` 폴더의 모든 파일을 `C:\extension\`에 복사

---

## 2단계: 크롬 확장 프로그램 로드

1. Chrome 주소창에 `chrome://extensions/` 입력
2. 우측 상단 **"개발자 모드"** 토글 켜기
3. **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. 폴더 선택: `C:\extension`

---

## 3단계: Native Messaging Host 설치

1. `C:\extension\install_host.bat`을 **관리자 권한으로** 실행 (우클릭 → 관리자 권한으로 실행)
2. `[OK] Manifest`, `[OK] Registry done` 메시지 확인
3. Enter 눌러 종료

---

## 4단계: Chrome 재시작

- Chrome을 **완전히 종료** 후 다시 열기
- (탭 닫기가 아니라 완전 종료)

---

## 5단계: 확장 프로그램 설정

1. Chrome 우측 상단 퍼즐 아이콘 클릭
2. "채널에이전트" 옆 점 세 개 클릭 → **옵션**
3. **"Google 로그인"** 버튼 클릭 → 새 탭이 열림 → `kangmj980124@gmail.com` 계정으로 로그인
4. 로그인 후 카카오 비즈니스 페이지로 자동 이동되면 성공
5. 다시 옵션 페이지로 돌아와서 **거래처 폴더 경로** 입력 (예: `C:\Users\taxju\OneDrive\Desktop\GDRIVE\거래처`)
6. **"저장하기"** 클릭

---

## 6단계: 동작 확인

1. https://business.kakao.com/ 접속
2. 상담 목록 페이지 → 사이드바 할일 목록이 나타나면 정상
3. 채팅 페이지 → AI 버튼들이 나타나면 정상

---

## 업데이트 시

정교가 deploy 후 알려주면:
1. GitHub에서 최신 코드 pull
2. `C:\extension\`에 파일 덮어쓰기 복사
3. `chrome://extensions/` → 채널에이전트 새로고침 버튼 클릭

---

## 문제 발생 시

| 증상 | 해결 방법 |
|------|----------|
| 크롬 재시작 시 확장 삭제됨 | Google Drive가 아닌 `C:\extension\`에서 로드했는지 확인 |
| AI 버튼 안 나옴 | 확장 프로그램이 활성화돼 있는지 확인 |
| 로그인 실패 | 옵션 → 로그아웃 후 다시 Google 로그인 |
| 할일 안 불러와짐 | 로그인 상태 확인 (옵션 페이지에서 초록색 = 정상) |
| 폴더 열기 안 됨 | `C:\extension\install_host.bat` 관리자 권한으로 다시 실행 |
