채널에이전트 설치 가이드 (문정 PC)

사전 조건

  • Google Drive 동기화가 되어 있어야 합니다 (extension 폴더 공유됨)
  • Chrome 브라우저가 설치되어 있어야 합니다

1단계: 크롬 확장 프로그램 로드

  1. Chrome 주소창에 chrome://extensions/ 입력
  2. 우측 상단 "개발자 모드" 토글 켜기
  3. "압축해제된 확장 프로그램을 로드합니다" 클릭
  4. 폴더 선택: C:\Users\taxju\OneDrive\Desktop\GDRIVE\projects\extension
  5. 확장 프로그램 ID가 febheajklmkbakaolobeamnffdakoida인지 확인

2단계: Native Messaging Host 설치

  1. 파일 탐색기에서 C:\Users\taxju\OneDrive\Desktop\GDRIVE\projects\extension 폴더 열기
  2. install_host.bat 더블클릭
  3. 창이 뜨고 자동으로 설치 완료됨

3단계: Chrome 재시작

탭 닫기가 아니라 Chrome을 완전히 종료 후 다시 열어야 합니다.

4단계: 확장 프로그램 설정

  1. Chrome 우측 상단 퍼즐 아이콘 클릭
  2. "채널에이전트" 옆 점 세 개 클릭 → 옵션
  3. "Google 로그인" 버튼 클릭 → kangmj980124@gmail.com 계정으로 로그인
  4. 거래처 폴더 경로 입력 (예: C:\Users\taxju\OneDrive\Desktop\GDRIVE\거래처)
  5. "저장하기" 클릭

5단계: 동작 확인

  1. https://center-pf.kakao.com/ 접속
  2. 채팅 페이지 → AI 버튼들이 나타나면 정상
  3. 상담 목록 페이지 → 사이드바 할일 목록이 나타나면 정상
